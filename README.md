# 🚀 Production-Ready EKS Deployment

A real-world DevOps project demonstrating production-grade Kubernetes deployment on AWS EKS with zero-downtime strategies, automated scaling, and safe release management.

---

# 📌 Overview

This project demonstrates how to deploy and manage containerized applications using Kubernetes on AWS EKS. It focuses on reliability, scalability, and production deployment strategies such as Rolling Updates and Blue/Green deployments.

---

# 🧰 Tech Stack

- AWS EKS (Kubernetes)
- NGINX Docker Image
- kubectl
- eksctl
- AWS Load Balancer

---

# ⚙️ Key Features

- Rolling Updates with zero downtime
- Health checks (Liveness & Readiness Probes)
- Blue/Green deployment strategy
- Load Balancer exposure
- Automatic rollback support
- Scalable node group configuration

---

# 🔵🟢 Deployment Strategy

The project uses Blue/Green deployment to ensure safe production releases:

- Blue = Current stable version
- Green = New version for testing
- Traffic switching via Kubernetes Service selector

---

# 🔄 Reliability Features

- Automatic pod health validation
- Self-healing containers
- Rolling updates with controlled rollout
- Instant rollback capability

---

# 📊 What This Project Demonstrates

- Production-grade Kubernetes architecture
- Real-world deployment workflows
- DevOps best practices
- High availability system design on AWS

---

# 🚀 Outcome

A fully functional cloud-native deployment system with safe deployment strategies, traffic management, and scalable infrastructure on AWS EKS.
